# 智伴口袋

> 您的个人专家资料库小程序

## 功能特性

### 🎯 专家库
- 个人名片展示与编辑
- 技能管理与熟练度评估
- 项目经验记录
- 知识卡片管理

### 🤖 AI助手
- 智能问答
- 学习建议
- 代码辅助

### 📊 数据管理
- 本地数据存储
- 数据备份与恢复
- 统计概览

## 项目结构

```
zhibankoudai/
├── miniprogram/           # 小程序源码
│   ├── app.js            # 应用入口
│   ├── app.json          # 应用配置
│   ├── app.wxss          # 全局样式
│   ├── pages/            # 页面目录
│   │   ├── index/        # 首页
│   │   ├── expert/       # 专家库
│   │   ├── ai/           # AI助手
│   │   └── mine/         # 我的
│   ├── utils/            # 工具函数
│   └── images/           # 图片资源
├── cloudfunctions/       # 云函数目录
└── project.config.json   # 项目配置
```

## 快速开始

1. 克隆项目到本地
2. 使用微信开发者工具导入项目
3. 设置小程序的 AppID
4. 编译运行

## 技术栈

- 微信小程序原生框架
- WXML + WXSS + JavaScript
- 云开发能力（可选）
